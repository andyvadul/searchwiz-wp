# Getting Started with SearchWiz

This guide will help you install and configure SearchWiz on your WordPress site.

## Installation

### From WordPress Plugin Directory

1. Log in to your WordPress admin dashboard
2. Go to **Plugins → Add New**
3. Search for "SearchWiz"
4. Click **Install Now**
5. Click **Activate**

### Manual Installation

1. Download the plugin zip file from [GitHub](https://github.com/andyvadul/searchwiz-wp/releases)
2. Go to **Plugins → Add New → Upload Plugin**
3. Choose the zip file and click **Install Now**
4. Click **Activate**

## Initial Setup

After activation, SearchWiz works immediately with sensible defaults. To customize:

1. Go to **Settings → SearchWiz**
2. Configure your preferences (see sections below)
3. Click **Save Changes**

## Configuration Options

### General Settings

- **Enable Ajax Search** - Turn on/off instant search results
- **Minimum Characters** - How many characters before search starts
- **Results Per Page** - Number of results to show

### Display Settings

- **Highlight Search Terms** - Highlight matching text in results
- **Show Thumbnails** - Display featured images in results
- **Result Layout** - Choose list or grid layout

### Advanced Settings

- **Post Types to Search** - Select which content types to include
- **Exclude Categories** - Hide specific categories from search

## Testing Your Setup

1. Go to your site's frontend
2. Use the search box in your header/widget
3. Start typing to see instant results
4. Click a result to verify navigation works

## Troubleshooting

If search isn't working:

1. Check **Settings → SearchWiz → Debug Mode** is off (unless troubleshooting)
2. Clear your browser cache
3. Clear any caching plugin cache
4. Check for JavaScript errors in browser console

See [Troubleshooting Guide](troubleshooting.md) for more help.
